# Excursie Berlijn App

Mobiele excursie-app voor de ontwikkelweek.

## Online website

Voeg hier de live URL van de website toe zodra die online staat:

`https://jouw-url-hier.nl`

## Database

1. Importeer [database.sql](database.sql) in MySQL.
2. Gebruik de database `excursie_berlijn_app`.
3. Controleer of XAMPP / MySQL en PHP actief zijn.

## Wat zit erin

- Homepagina met uitgelichte Berlijn-content.
- Bezienswaardigheden uit de database.
- Activiteiten en tips uit de database.
- Praktische informatie voor onderweg.
- Zoekresultaten op basis van database-content.

## Belangrijk

De CMS-pagina's en inlogfunctionaliteit zijn nog niet ingevuld. De frontend is nu wel gekoppeld aan de database voor het tonen van content.
